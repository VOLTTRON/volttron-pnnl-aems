Welcome to AEMS

Advanced Energy Management System

This is the default welcome page for AEMS deployments. To customize this page for your specific deployment:

  1. Create a new directory under /static/templates/ named {CAMPUS}-{BUILDING} using all lowercase letters
  2. Copy this welcome-content.html file to your new directory
  3. Customize the content as needed
  4. Set the environment variables:
    • VOLTTRON_CAMPUS - Your campus identifier
    • VOLTTRON_BUILDING - Your building identifier

Template System Features

  • No Rebuild Required: Edit HTML files directly without rebuilding the application (the Docker image will need to be rebuilt)
  • Environment-Driven: Automatically selects templates based on VOLTTRON_CAMPUS and VOLTTRON_BUILDING
  • Shared Styling: Inherits CSS from the main application
  • Easy Deployment: Simply copy template folders for new sites

Directory Structure

/static/templates/
├── shared/
│   └── styles.css               # For preview/testing only
├── default/                     # Fallback templates
│   ├── welcome-content.html     # Content injected in app
│   ├── welcome-preview.html     # Standalone test page
│   ├── info-content.html
│   └── info-preview.html
└── {CAMPUS}-{BUILDING}/        # Deployment-specific
    ├── welcome-content.html
    ├── welcome-preview.html
    ├── info-content.html
    └── info-preview.html

Getting Started

This template system makes it easy to customize welcome and info pages for different AEMS deployments. Each deployment can have unique content while maintaining consistent styling across all sites.

Note: This content fragment inherits styling from the main application. For standalone testing, use the corresponding -preview.html file.
